一、参数解释
【LTE】
EARFCN - 频点
BAND - 频段
PCI - 物理小区识别码，右上角数字代表模三后的值
RSRP - 参考信号接收功率
RSRQ - 参考信号接收质量
SINR - 信号与干扰加噪声比
PRI - 小区优先级
P - 主小区
S - 副载波小区（如果基站开通了载波聚合）
N - 邻区
MCC/MNC- 国家代码网络代码
Operator-运营商
CP Type - 循环前缀类型（分普通和扩展2种）
Duplex Mode - 双工模式（分TDD,FDD 2种）
TAC - 路由区跟踪码
ECI -基站标识（有eNodeb和cell id组成）
BandWidth-带宽（LTE最大20MHz）
Frequency-频率
Antenana Tx/Rx - 基站收发天线数目
Timging advance- 时间提前量（和当前基站的距离）
CA Active Status -载波聚合活动状态
P-a - PDSCH 功率调整参数A
P-b - PDSCH 功率调整参数B，通常在SIB2配置
PRACH Txpower - 随机接入信道发送功率
PUSCH Txpower - 上行共享信道发送功率（上传功率）
Rank Indicator - 秩指示
Transmission Mode - 传输模式（不同传输模式和层数有关系）
CQI - 信道质量指示
Pathloss- 路径耗损（设备到基站）
TDD Configuration - TDD帧与特殊子帧配置
TDD UL/DL Configuration - TDD上下行切换周期
MCS - 编码调制方案（0-31）
RBs - 传输资源块数量（0-100）
QPSK - 正交相移键控（调制方式）
16QAM - 16种符号的正交幅度调制（调制方式）
64QAM - 64种符号的正交幅度调制（调制方式）
256QAM - 256种符号的正交幅度调制（调制方式）
BLER - 误块率
APN - 存取接入点
PDN IP - 数据包网络IP地址
QCI - 服务质量（QoS）级别
AMBR DL - 签约用户最大下行速率
AMBR UL - 签约用户最大上行速率
APP Tput - 应用层速率
PDCP Tput - PDCP（分组数据汇聚协议层）速率
RLC Tput - RLC（无线链路层）速率
MAC Tput - MAC（媒体接入层）速率
PHY Tput - PHY（物理层）速率
Modulation - 调制方式
RACH - 随机接入
C_RNTI - 小区无线网络临时标识（RRC连接态下有效）
RA-RNTI - 随机接入临时标识
RACH Procedure type- 随机接入过程类型（基于竞争的还是自由模式）
RACH Procedure mode - 随机接入原因
【高清语言】
RTP - 实时传输协议
RTP Response Status Code - 响应码
RTP Direction - 传输方向（上行，下行）
RTP Sequence Number - 序列号
RTP Timestamp - 时间戳
RTP Source SSRC - 源同步标识
RTP Loss Type - 丢包类型
RTP Loss Rate - 丢包率
RTP Throughput - 吞吐率
RTP Codec Type - 编码类型
RTP Media Type - 媒体类型
RTP Audio CMR - 音频编码方式
RTP Audio Frame Content - 音频帧大小
RTP Loss Num - 丢包数量
RTP Payload size- 承载内容大小
ROHC Enabled - 是否支持头部压缩
【NR】
ARFCN - 频点
BEAM - 波束赋型
S - 服务小区
N - 邻区
5G State - 5G 接入是否受限制（指得设备）
5G Available -5G 网络是否存在
RRC State RRC - 状态，初始化，空闲，连接等
Connectivity Mode - 连接模式：SA/NSA
gNodeB/CellId - 基站和小区编号
ServingCellIndex - 服务小区索引编号
PointA Arfcn - POINT A 频点
Numerology - 子载波间隔类型（0，1，2，3，4）
SubCarrierSpacing - 子载波带宽（15，30，60，120，240）
Q Rxlev Min - 驻留在该小区的最小接收电平（需要换算）
LocationAndBandWidth - BWP 的频域位置和带宽
SSB Arfcn/Freq DL - SSB 的频点和频率
Band/BandWidth DL - 频段及带宽
Center Arfcn/Freq DL - 中心频点及下行频率
SSB GSCN SSB - 绝对频域位置
SSB Beam Index - 服务小区当前波束索引
SubCarrierOffset - SSB 与 Point A 的偏移
SS RSRP - SSB 电平值
SS RSRQ - SSB 参考信号质量
SS SINR - SSB 信噪比
PUSCH TXPOWER - 上行共享数据信道传输功率
PUCCH TXPOWER - 上行控制信道传输功率
PRACH TXPOWER - 随机接入信道传输功率
MM STATE - 移动性管理状态（NAS 层），分注册和去注册2 个状态
MM SUB STATE - 移动性管理子状态（NAS 层）
AMF IDENTIY - 接入和移动性管理标识,属于 RAN
5G TMSI - 5G 临时识别码
DNN - 同 APN,接入点编号
RACH REASON - 随机接入的原因（连接请求、切换，上行数据到达）
RACH CONTENTION - 随机接入竞争类型（FREE,DL MCE,UL GRANT,PDCCH C-RNTI）
PRACH CONFIG - 随机接入配置
SSB ID - 发起随机接入的 BEAM 索引
RACH PREAMBLE FORMAT - MSG1 PREAMBLE FORMAT(0,1,2,3,A1,A2,A3…)
RACH LOGICAL ROOT INDEX - 根序列索引
RACH RA RNTI - 随机接入临时码
RACH ACCESS RESULT - 随机接入结果
【GSM】
LAC - 位置区码
CI - 小区编码
BCCH - 广播信道
BSIC - 基站识别码
C1 - 小区选择和重选参数1
C2 - 小区选择和重选参数2
RxLev - 接收电平值
S - 服务小区
N - 邻小区
RxLevBCCH - 广播信道接收电平
DTX - 非连续传输模式
RxLevFul - 接收信号强度
RxLevSub - 接收信号强度（开通DTX）
RxQualFul - 通话质量
RxQualSub - 通话质量（开通DTX）
TCH ARFCN - 业务信道频点
TCH C/I - 业务信道载干比
RLT Value - 无线链路超时值
Channel Mode - 信道模式
Hopping Mode - 跳频模式（是否支持）
Hopping List - 调频列表
HSN - 跳频序列号
MAIO - 移动分配索引偏置
Speech Codec DL - 下行语言编码
Speech Codec UL - 上行语言编码
LLC - 逻辑链路控制
RLC - 无线链路控制
MAC - 媒体接入控制
【WCDMA】
UTRAN - UMTS无线接入网
Carrier - 载波（主载波，副载波）
Cell - 小区类型（活动集，监测集和候选集）
UARFCN - 频点
SC - 扰码
RSCP - 导频信道信号强度
URA - UTRAN 注册区域
PSC - 主扰码
RSSI - 接收信号强度指示
EC/IO - 接收信号的强度和邻小区干扰水平的比值
ActiveSet - 活动集
MonitorSet - 监测集
DedicateSet- 候选集
RRC State - 无线资源控制
SIR - 信号噪声比
PCA - 功率控制算法
TPC Step size - 传输功率控制步长大小
AICH Timing - 采集指示信道时延
Preamble Offset - 前导码偏移量
Preamble Count - 前导码数量
GMM State - GPRS移动性管理
MM State - 移动性管理状态
TB Size - 传输块大小
PDU - 协议数据单元
SDU - 服务数据单元
